# Peers

## Get Peers

```shell
curl "http://127.0.0.1:9085/peers"
```

```http
GET peers HTTP/1.1
Host: 127.0.0.1:9085
```

Returns an array of all the IP's of the peers to which the application is currently connected.

### REQUEST

`GET peers`
